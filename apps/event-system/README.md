# Event System

イベント運営システム - 企業と学生のマッチングイベントを管理・運営するアプリケーション

## ポート番号

- 開発環境: `http://localhost:3001`

## 起動方法

```bash
# monorepoルートから
pnpm --filter event-system dev

# または、event-systemディレクトリで
pnpm dev
```
