# Agent Manager

エージェント管理システム - 候補者、企業、求人、応募の管理を行うアプリケーション

## ポート番号

- 開発環境: `http://localhost:3002`

## 起動方法

```bash
# monorepoルートから
pnpm --filter agent-manager dev

# または、agent-managerディレクトリで
pnpm dev
```
